Two dialects, one core

A plain import forterp is strict ANSI X3.9-1966 — it rejects non-F66 constructs, not merely runs standard ones. Flip to FORTRAN10 for the DEC superset: octal, Hollerith, IMPLICIT, apostrophe strings, random-access I/O. Eighteen gated knobs separate the two.

Pluggable machine target

The value model is a Target: word width and overflow, the logical-truth convention, the character codec. NATIVE (64-bit, ASCII) is the portable default; PDP10 is the faithful 36-bit, 5×7-bit packed-ASCII, .TRUE.=−1 DEC machine.

Validated against real code

The PDP-10 target was extracted from an interpreter that runs the 1978 multiplayer game DECWAR and Walter Bright's Empire unmodified — gnarly period source, not toy snippets — and checked against the DEC FORTRAN-10 V5 manual and the ANSI FCVS conformance corpus.
