Two dialects, one core

A plain import forterp is strict ANSI FORTRAN 66 — it rejects non-F66 constructs, not merely runs standard ones. Flip to FORTRAN10 for the DEC superset: octal, Hollerith, IMPLICIT, apostrophe strings, random-access I/O. Eighteen gated knobs separate the two.

Pluggable machine target

The value model is a Target: word width and overflow, the logical-truth convention, the character codec. NATIVE (64-bit, ASCII) is the portable default; PDP10 is the 36-bit, 5×7-bit packed-ASCII, .TRUE.=−1 DEC machine (integers and packing exact; REAL is a host double, so floats diverge at the rounding/exponent edges).

Validated against real code

The PDP-10 target was exercised against real 1970s DEC FORTRAN — full programs, not toy snippets — and checked against the DEC FORTRAN-10 V5 manual and the FCVS conformance corpus.
